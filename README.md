üñ•Ô∏è How It Works

- The quiz loads a set of predefined questions.
- Each question displays multiple answer options.
- The user selects an answer.
- The app checks if the answer is correct and updates the score.
- The next question is displayed using the next button.
- The previous button can take you to the previous question.
- At the end of the quiz, the final score is shown.
